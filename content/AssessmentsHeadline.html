Having problems? Please make sure you are using the Google Chrome browser.

This is our officially supported browser and is freely available for all computer systems

We are aware of an issue causing uploads using the Safari browser to freeze - please use the Google Chrome browser to avoid this issue

Internet Explorer is particularly problematic and might cause some parts of the page to be hidden.

If you are using an NHS PC, Chrome should be installed but you might need to look in the Applications menu to find it.
